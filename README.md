# Light Simulator

---

### Intro

No wires: light transmitted through air! 

Only **One** basic logic circuit building blocks: NOR Gate.

Use NOR Gate to build a CPU!

Other blocks serve as light reflectors and splitters.


---

### Requirements

    Tkinter

### Usage

 1. Clone this repo.
 2. run `python3 GUI.py`
 3. Enjoy hacking!
